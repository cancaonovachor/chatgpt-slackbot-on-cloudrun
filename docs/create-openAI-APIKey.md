# OpenAI APIトークン取得手順書

この手順書では、OpenAI APIトークンの取得方法を説明します。

## 1. OpenAIアカウントの作成

1. [OpenAIのウェブサイト](https://www.openai.com/)にアクセスし、右上の**Sign up**ボタンをクリックしてアカウントを作成します。

2. 必要な情報を入力し、アカウント作成手順に従ってアカウントを登録します。

## 2. APIトークンの取得

1. [OpenAI Dashboard](https://platform.openai.com/signup)にログインします。

2. 左側のメニューから**API Keys**を選択します。

3. **Create API Key**ボタンをクリックして、新しいAPIキーを作成します。

4. 作成されたAPIキーは、**Key**欄に表示されます。このキーをコピーし、安全な場所に保管してください。

**注意**: APIキーは秘密情報です。他人に漏らさないようにし、信頼できる環境でのみ使用してください。

以上で、OpenAI APIトークンの取得が完了です。
